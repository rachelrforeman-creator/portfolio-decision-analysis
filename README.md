# portfolio-decision-analysis

This repository contains a Jupyter notebook demonstrating
portfolio optimization under budget and downside risk constraints
using synthetic data.

## What this notebook demonstrates
- Risk-adjusted NPV (rNPV) valuation
- Monte Carlo uncertainty analysis (P10 / P50)
- Portfolio selection under budget and downside constraints
- Visualization of cost, value, and risk trade-offs

## Files
- `Rare_Disease_Feasibility_Portfolio_Selection_CLEAN.ipynb`  
  Main notebook containing the full analysis and results.

## Notes
- All data is synthetic and illustrative
- This project demonstrates methodology only, not real assets
